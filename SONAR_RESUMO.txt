╔════════════════════════════════════════════════════════════════╗
║                    OPÇÃO 1: SONARCLOUD                        ║
╚════════════════════════════════════════════════════════════════╝

Setup (10 minutos):

1. Acesse: https://sonarcloud.io
2. Faça login com GitHub
3. Autorize o acesso a seus repos
4. Crie um novo projeto
5. Copie o SONAR_TOKEN
6. GitHub > Settings > Secrets > New secret
   Nome: SONAR_TOKEN
   Valor: [cole o token]
7. Edite .sonarcloud.properties:
   sonar.organization=seu-usuario-ou-org
   sonar.projectKey=tempo-justo-api
8. Faça um git push
9. ✅ O workflow CI/CD rodará automaticamente!

Análise automática em:
• Cada push para main/develop
• Cada Pull Request

Visualizar resultados em:
• https://sonarcloud.io/dashboard

╔════════════════════════════════════════════════════════════════╗
║                OPÇÃO 2: SONARQUBE LOCAL                       ║
╚════════════════════════════════════════════════════════════════╝

Setup (10 minutos):

1. Inicie o Docker:
   docker-compose -f docker-compose.sonar.yml up -d

2. Acesse:
   http://localhost:9000

3. Faça login:
   Usuário: admin
   Senha: admin

4. Crie um novo projeto
5. Gere um token
6. Execute:
   export SONAR_TOKEN=seu-token
   ./scripts/sonar-local.sh

7. ✅ Análise pronta em http://localhost:9000
